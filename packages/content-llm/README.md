# @statewalker/content-llm

Content-aware LLM adapters: messages, reasoning, skills, MCP tools — all speaking `@statewalker/content-blocks`.

## Installation

```sh
pnpm add @statewalker/content-llm
```

## Sub-exports

- `./messages` — block ↔ LLM message marshalling.
- `./reasoning` — reasoning-trace utilities.
- `./skills` — skill registration + dispatch.
- `./mcp` — Model Context Protocol adapters.
- `./tools` — Vercel AI-SDK tool helpers.

## Related

- `@statewalker/content-blocks`, `@statewalker/ai-provider`.
